# 01 - 项目愿景 (Project Vision)

## 一句话定义

一个可复用的 LÖVE2D 2D 游戏引擎框架，首作为中国象棋，面向 Steam/桌面/APK。

## 产品定位

- **类型**：休闲/棋牌，带轻度视觉打磨
- **风格**：2D 俯视偏斜角（约 15° 俯角偏移），木纹 + 玉石质感，暖色调
- **目标用户**：喜欢象棋的休闲玩家、独立游戏技术社区
- **定价区间**：Steam 上做免费 / 低价 + 可选 DLC（后续扩展残局包等）
- **单机为主**：第一版不做联机，纯本地

## 视觉方向 — "2.1D 伪纵深"

借鉴产品：Monument Valley（分层错觉）、Into the Breach（俯视微透视）、杀戮尖塔（卡牌阴影与层次）。**绝不抄其具体美术资源，只借鉴"为什么看起来舒服"。**

具体手段：
| 手段 | 效果 | 实现阶段 |
|------|------|---------|
| 棋盘加斜视角 15° | 立体感 | V1 |
| 棋子加投影椭圆 | 棋子"浮"在棋盘上 | V1 |
| 棋盘边缘加厚度（侧面深色矩形） | 像真实棋盘有边 | V1 |
| 粒子（落子灰尘、吃子光效） | 反馈感 | V2/V4 |
| 缓动动画（easeOutBack 等） | 落子弹性感 | V2 |
| 镜头微震（吃子/将军时） | 力度反馈 | V3 |
| 简单 shader（径向光、水面光斑） | 氛围 | V4 |

> 明确不做：OpenGL 3D 相机、Z 轴深度测试、法线贴图、骨骼 3D。

## 非目标 (Non-goals)

- V0-V3 不做联机
- V0-V3 不做国际化（先中文 + 英文预留字符串 key）
- V0-V5 不做付费 DLC
- 不做 Web 版（LÖVE 支持 WebAssembly 但性能和打包坑多，优先级低）
- 不做 iOS（Apple 开发者账号 $99/年 + 审核风险，APK 阶段先验证）
